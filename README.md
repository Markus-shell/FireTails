# 🦊 FireTails (Fenix Rebranded)

**FireTails** is a fast and private Android web browser, built on the official source code of **Mozilla Fenix** (Firefox for Android).

This project is a custom fork aimed at rebranding, streamlining, and optimizing the original browsing experience.

---

## 🔄 Main Changes

- **Complete Rebranding**: Transformed the visual and textual identity from *Mozilla Fenix* / *Firefox* to **FireTails**.
- **Cleanup**: Prepared the codebase for integrating custom features and disabled unnecessary telemetry modules.
- **Modern Core**: Preserves the power of the original architecture built on **Kotlin**, **GeckoView** (Mozilla's rendering engine), and **Android Components**.

---

## 🛠️ Prerequisites to Build FireTails

To open and compile this project on your machine:

- **Android Studio**: A recent stable version (e.g., Ladybug / Koala).
- **Java Development Kit (JDK)**: **JDK 17** (required for Gradle compatibility).
- **Android SDK**: SDK 33 or higher.
- **Memory**: At least 8 GB of RAM recommended to handle compilation.
